The Hash Table Advantage

                        

                            Sets and Dictionaries utilize hashing mechanisms. This means Python computes a numeric hash for the data, instantly directing it to a specific memory bucket. Therefore, looking up data takes exactly the same amount of time ($O(1)$) whether there are 10 items or 10 billion.
                        

                    

                    

                        
The Linear Scan Tax

                        

                            Lists and Tuples are sequential arrays. If you search for an item, Python must start at index 0 and inspect every single item one by one until it finds a match. This is Linear Time ($O(N)$), meaning time scales directly with the amount of data.
                        

                    

                

            

        


        

            

                
Comprehensions & Evaluation Paradigms

                

                    Comprehensions flatten verbose loops into elegant single lines. In this section, explore how identical syntax logic produces entirely different memory behaviors (Eager vs. Lazy evaluation) simply by changing the enclosing brackets.
                

            


            

                

                    
High Memory

                    

                        [] List Comprehension
                    

                    
Eager Evaluation

                    

                        squares = [x**2 for x in range(3)]
print(squares)
                    

                    

                        Output: [0, 1, 4]
                    

                    

                        Evaluates all items instantly. The CPU crunches every iteration upfront and stores the fully populated array structure directly into active RAM. Highly readable, but crashes systems if the dataset approaches gigabytes.
                    

                


                

                    
Low Memory

                    

                        () Generator Comprehension
                    

                    
Lazy Evaluation

                    

                        gen = (x**2 for x in range(3))
print(next(gen))
                    

                    

                        Output: 0
                    

                    

                        Evaluates items on-demand. Creates an iterator object that pauses execution. Data is generated strictly when `next()` is called. Maintains a nearly zero memory footprint, making it perfect for streaming massive files.
                    

                


                

                    

                        {} Set Comprehension
                    

                    

                        uniques = {x.lower() for x in ["A", "a", "B"]}
print(uniques)
                    

                    

                        Identical internal logic to lists, but enforces the strict mathematical Set rules dynamically. As it eagerness processes the data, it instantly filters out any duplicates generated by the expression.
                    

                


                

                    

                        {k:v} Dict Comprehension
                    

                    

                        squares = {x: x**2 for x in range(3)}
print(squares)
                    

                    

                        Constructs key-value mappings dynamically. The expression is split by a colon. Excellent for swapping keys and values, or generating relationship mappings from arrays.
                    

                

            

        


        

            

                
Methods & Built-in Functions

                

                    A comprehensive cheat sheet of the most heavily tested built-in functions and type-specific methods. This covers array mutations, string string processing, set mathematics, and dictionary key-value traversals.
                

            


            

                

                    
                        

                            Type
                            Method / Function
                            Description
                            Syntax Example
                        

                    
                    
                        
                        
                        

                            List
                            .append(item)
                            Adds a single item to the very end of the list.
                            l.append(5)
                        

                        

                            List
                            .extend(iterable)
                            Unpacks an iterable and adds all its items to the end.
                            l.extend([6, 7])
                        

                        

                            List
                            .insert(index, item)
                            Inserts an item at a specific index position, shifting others.
                            l.insert(0, "start")
                        

                        

                            List
                            .remove(item)
                            Removes the first occurrence of the specified item.
                            l.remove("apple")
                        

                        

                            List
                            .pop([index])
                            Removes and returns the item at the index (defaults to the last).
                            val = l.pop(1)
                        


                        
                        

                            String
                            .upper() / .lower()
                            Returns a new string converted entirely to uppercase or lowercase.
                            "Hi".upper()
                        

                        

                            String
                            .strip()
                            Removes leading and trailing whitespace from the string.
                            " hi ".strip()
                        

                        

                            String
                            .split(separator)
                            Splits a string into a list based on a provided separator.
                            "a,b".split(",")
                        

                        

                            String
                            .join(iterable)
                            Joins elements of an iterable using the string as a separator.
                            "-".join(["a", "b"])
                        


                        
                        

                            Set
                            .add(item)
                            Adds a single element to the set uniquely.
                            s.add(10)
                        

                        

                            Set
                            .discard(item)
                            Removes element. Safely ignores the command if item is missing.
                            s.discard(99)
                        

                        

                            Set
                            .remove(item)
                            Removes element. Raises KeyError if item is missing.
                            s.remove(10)
                        


                        
                        

                            Tuple
                            .count(item)
                            Returns the number of times an item appears in the tuple.
                            t.count(5)
                        


                        
                        

                            Dict
                            .keys() / .values()
                            Returns a view object of all keys or all values in the dict.
                            d.keys()
                        

                        

                            Dict
                            .items()
                            Returns a view object of all (key, value) tuple pairs.
                            d.items()
                        

                        

                            Dict
                            .get(key, default)
                            Safely retrieves value for a key; returns default if missing.
                            d.get("age", 0)
                        

                        

                            Dict
                            .pop(key)
                            Removes the specified key and returns its value.
                            val = d.pop("id")
                        


                        
                        

                            Built-In
                            len(iterable)
                            Returns the total number of items in the collection.
                            len([1, 2, 3])
                        

                        

                            Built-In
                            sorted(iterable)
                            Returns a brand new sorted list from the iterable elements.
                            sorted({"c":3, "a":1})
